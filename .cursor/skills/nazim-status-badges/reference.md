# Status Badge Reference

## Common Status Mappings

### Student Status
| Status | Variant |
|--------|---------|
| active | default |
| admitted | secondary |
| applied | outline |
| withdrawn | destructive |
| graduated | outline |

### Staff Status
| Status | Variant |
|--------|---------|
| active | default |
| inactive | secondary |
| on_leave | outline |
| terminated | destructive |
| suspended | destructive |

### Fee/Payment Status
| Status | Variant | Color |
|--------|---------|-------|
| paid | default | green |
| partial | secondary | yellow |
| pending | outline | orange |
| overdue | destructive | red |

### Attendance Status (use statusOptions with color)
| Status | Color |
|--------|-------|
| present | green |
| absent | red |
| late | yellow |
| excused | blue |
| sick | purple |
| leave | orange |

### DMS Document Status
| Incoming | Outgoing |
|----------|----------|
| pending | draft |
| under_review | issued |
| completed | printed |

### Online Admission Status
| Status | Variant |
|--------|---------|
| accepted | default |
| rejected | destructive |
| under_review | secondary |
| archived | outline |
