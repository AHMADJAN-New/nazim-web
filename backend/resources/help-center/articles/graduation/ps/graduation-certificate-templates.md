# د فراغت تصدیق لیک ټیمپلیټونه

د تصدیق لیک ټیمپلیټونو پاڼه هغه ځای دی چیرته تاسو د فراغت بست څخه تصدیق لیکونه صادرولو کې کارول شوي فراغت تصدیق لیک ټیمپلیټونه جوړوئ او مدیریت کړئ. د مکتب مدیران او کارکوونکي دا پاڼه کاروي ترڅو ټیمپلیټونه اضافه کړي (سرلیک، مکتب، توضیح، د پاڼې اندازه، RTL، فعال حالت)، د شاتړ توافق لپاره اختیاري body HTML وټاکي، او د Layout Editor وکاروي ترڅو د تصدیق لیک عناصر (لکه د زده کوونکي نوم، نیټه، لوګو) د drag-and-drop سره موقعیت او سټایل کړي. هر ټیمپلیټ کیدی شي ځانګړي مکتب ته ورکړل شي یا د ټولو مکتبونو لپاره عمومي ټیمپلیټ پاتې شي.

---

## د پاڼې لنډیز

کله چې تاسو د تصدیق لیک ټیمپلیټونو پاڼه خلاصه کړئ، به ووینئ:

### معلومات جدول

اصلي جدول ټول فراغت تصدیق لیک ټیمپلیټونه له دې کالمونو سره لیست کوي:

| کالم | توضیح |
|------|--------|
| سرلیک | د ټیمپلیټ نوم/سرلیک. |
| مکتب | دا ټیمپلیټ چې مکتب ته ورکړل شوی، یا که هیڅ مکتب ټاکل شوی نه وي "ټول مکتبونه" (عمومي ټیمپلیټ). |
| شالید | "هو" (د آیکون سره) که شالید انځور ټاکل شوی وي؛ "هیڅ" په نورو. |
| حالت | نښان: فعال یا غیر فعال. |
| جوړ شوی | د ټیمپلیټ د جوړیدو نیټه. |
| عملیات | د layout سمون (د layout آیکون) که ټیمپلیټ layout config ولري؛ حذف (د کڅوړې آیکون). |

### د قطار عملیات

- **د Layout سمون** — د دې ټیمپلیټ لپاره د Layout Editor ډیالوګ خلاصوي. تاسو کولی شئ عناصر وګاډوئ او موقعیت او سټایل وټاکئ او layout ساتئ. یوازې کله ښودل کیږي چې ټیمپلیټ layout config ولري.
- **حذف** — تایید ډیالوګ خلاصوي. تایید کول ټیمپلیټ تل پاتې حذف کوي. دا عمل بیرته اخیستل کیدی نشي.

---

## نوی ټیمپلیټ جوړول

د جدول لاندې، یو **جوړول** کارډ فورم لري. ساحې ډک کړئ او **ساتل** کلیک وکړئ.

| ساحه | ډول | اړین | توضیح |
|------|-----|------|--------|
| سرلیک | متن | هو | د ټیمپلیټ ښودل شوی نوم (لکه "د فراغت تصدیق لیک ۱۴۰۳"). |
| مکتب (اختیاري) | ټاکل | نه | ټیمپلیټ ځانګړي مکتب ته ورکړئ، یا د ټولو مکتبونو لپاره "هیڅ (عمومي ټیمپلیټ)" وټاکئ. |
| توضیح | متن ساحه | نه | د ټیمپلیټ لنډه توضیح (لکه هدف یا کال). |
| Body HTML (اختیاري) | متن ساحه | نه | د placeholders سره legacy HTML لکه `{{student_name}}`. یادونه وایی د drag-and-drop موقعیت لپاره Layout Editor وکاروئ. |
| د پاڼې اندازه | ټاکل | نه | A4 یا A5. اصلي A4. |
| RTL | سویچ | نه | ښي څخه چپ layout. اصلي په (true). |
| فعال | سویچ | نه | آیا ټیمپلیټ فعال او د تصدیق لیکونو صادرولو وخت کې شتون لري. اصلي په. |

### د ثبتولو وروسته څه کیږي

- ټیمپلیټ جوړیږي او په جدول کې ښکاري. بیا تاسو **د Layout سمون** خلاص کولی شئ (وروسته له هغه چې ټیمپلیټ layout config ولري) ترڅو د تصدیق لیک layout ډیزاین کړئ. د ټیمپلیټ جوړول اتوماتیک Layout Editor خلاصوي نه؛ د نوي قطار په **د Layout سمون** وکاروئ ترڅو که اړتیا وي layout او شالید شخصي کړئ.

---

## د ټیمپلیټ سمون (Layout Editor)

1. ټیمپلیټ په جدول کې ومومئ.
2. په عملیاتو کالم کې **د Layout سمون** (د layout آیکون) کلیک وکړئ. د Layout Editor ډیالوګ خلاصیږي.
3. ایډیټر تصدیق لیک د اوسني layout او اختیاري شالید انځور سره ښیي. تاسو کولی شئ عناصر وګاډوئ ترڅو موقعیت بدل کړئ او سټایل بدل کړئ (فونټونه، اندازې، alignment).
4. په ایډیټر کې **ساتل** کلیک وکړئ ترڅو layout ساتئ؛ ډیالوګ بندیږي او ټیمپلیټ تازه کیږي.
5. **لغوه** کلیک وکړئ یا ډیالوګ بند کړئ پرته له ساتلو.

هغه ټیمپلیټونه چې خوندي شوي layout config لري "د Layout سمون" ښیي؛ هغه چې ونلري ممکن یوازې حذف ښیي تر هغه پورې چې layout ساتل شي (لکه له بل بهیر څخه چې layout config جوړوي).

---

## د ټیمپلیټ حذف

1. په عملیاتو کالم کې د ټیمپلیټ لپاره **حذف** (کڅوړه) تڼۍ کلیک وکړئ.
2. تایید ډیالوګ ښکاري: "آیا تاسو ډاډه یاست چې دا تصدیق لیک ټیمپلیټ حذف کړئ؟ دا عمل بیرته اخیستل کیدی نشي."
3. ټیمپلیټ ساتلو لپاره **لغوه** کلیک وکړئ، یا د تل پاتې لرې کولو لپاره **حذف** کلیک وکړئ.

د ټیمپلیټ حذف کول هغه تصدیق لیکونه حذف نه کوي چې دمخه د هغه ټیمپلیټ په کارولو صادر شوي؛ یوازې ټیمپلیټ له راتلونکي کارونې څخه لرې کوي.

---

## د صادرولو اختیارونه

دا پاڼه PDF یا Excel صادرول نه ورکوي. د بست معلوماتو صادرولو لپاره د فراغت بستونو پاڼه وکاروئ. تصدیق لیک ټیمپلیټونه یوازې تنظیمات دي.

---

## لارښوونې او غوره کړنې

- **روان سرلیک وکاروئ** — مکتب نوم یا کال (لکه "فراغت ۱۴۰۳ - اصلي مکتب") شامل کړئ ترڅو کارکوونکي کولی شي صادرولو وخت کې سم ټیمپلیټ وټاکي.
- **کله چې اړتیا وي مکتب ورکړئ** — د یو مکتب لپاره ټیمپلیټ محدودولو "مکتب (اختیاري)" وکاروئ، یا د شریک ټیمپلیټ لپاره "ټول مکتبونه" پاتې کړئ.
- **Layout Editor وکاروئ** — د ثابت، بصري ډیزاین او placeholders لپاره د خام Body HTML پرځای Layout Editor غوره کړئ.
- **فعال/غیر فعال وټاکئ** — زوړ ټیمپلیټونه (غیر فعال) بند کړئ پرځای د حذف کولو که ممکن راتلونکي کې بیا صادرول یا مرجع ته اړتیا ولري.
- **د پښتو/عربي لپاره RTL** — هغه تصدیق لیکونو لپاره RTL وساتئ چې به نومونه او متن په ښي څخه چپ ژبو کې ښیي.
