# د ازموینې پاڼې ټیمپلیټونه (Exam Paper Templates)

د ازموینې پاڼې ټیمپلیټونه پاڼه (د ازموینې پاڼې) تاسو ته اجازه ورکوي د ازموینې پاڼې ټیمپلیټونه او د هغوی د رندر لپاره کارول شوي HTML ټیمپلیټ فایلونه اداره کړئ. تاسو په مکتب، ټولګي او مضمون پورې ټیمپلیټونه جوړوئ؛ ژبه، موده او اختیاري ټولې نمرې ټاکئ؛ او د layout لپاره اختیاري ټیمپلیټ فایل ضمیمه کړئ. پاڼه دوه ټبونه لري: یو د پاڼې ټیمپلیټونو لست لپاره (د فلټرونو، جدول، جوړول/سمون، د قطار عملیاتو او صادرولو سره) او یو د ټیمپلیټ فایلونو اداره کولو لپاره.

---

## د پاڼې لنډیز

کله چې تاسو دا پاڼه خلاصه کړئ، تاسو به ووینئ:

### لنډیز کارډونه

په دې پاڼه کې لنډیز کارډونه نشته.

### ټبونه

- **د ازموینې پاڼې** — د ازموینې پاڼې ټیمپلیټونو لست. فلټرونه، جدول، Create Template، Export او د قطار عملیات (مخکې کتنه، PDF جوړول، پوښتنې سمول، سمون، کاپي، حذف).
- **ټیمپلیټ فایلونه** — د ازموینې پاڼو layout لپاره کارول شوي HTML ټیمپلیټ فایلونه اداره کول (جوړول، سمول، حذف).

### فلټرونه او لټون (د ازموینې پاڼې ټب)

- **لټون** — د پاڼې سرلیک له لارې لټون.
- **مکتب** — د مکتب له مخې فلټر (ټول مکتبونه یا ځانګړی مکتب).
- **مضمون** — د مضمون له مخې فلټر (ټول مضمونونه یا ځانګړی مضمون).
- **ازموینه** — د ازموینې له مخې فلټر (ټولې ازموینې یا ځانګړی ازموینه). له ازموینې پرته پاڼې "Generic" ښیي.

---

## ډاټا جدول (د ازموینې پاڼې ټب)

اصلي جدول لاندې کالمونه ښیي:

| کالم | توضیحات |
|------|----------|
| سرلیک | د پاڼې ټیمپلیټ سرلیک. که د هغې ازموینې مضمون لپاره اصلي وي "Default" بج. |
| مضمون | مضمون (بج). |
| ازموینه | د ازموینې نوم (بج) یا "Generic". |
| ژبه | ژبه (انګلیسي، پښتو، فارسي، عربي). |
| موده | موده په دقیقو کې. |
| پوښتنې | په پاڼه کې د پوښتنو/آیټمونو شمیره. |
| نمرې | ټولې نمرې (محاسبه شوې یا ټاکل شوې). |
| حالت | فعال یا غیر فعال. |
| عملیات | د قطار عملیاتو لپاره ډراپ ډاون مینو (⋮). |

### د قطار عملیات

کله چې تاسو د عملیاتو مینو (⋮) باندې کلیک وکړئ، تاسو کولی شئ:

- **مخکې کتنه** — د ازموینې پاڼې مخکې کتنې ډیالوګ خلاصوي.
- **PDF جوړول** — د پاڼې د PDF په توګه جوړولو او ډاونلوډ ډیالوګ خلاصوي.
- **پوښتنې سمول** — د پاڼې سمون پاڼې ته وړي ترڅو په پاڼه کې پوښتنې اضافه، سم، ترتیب یا لرې کړئ.
- **سمون** — د ټیمپلیټ لپاره د سمون فورم خلاصوي (میټاډاټا، ژبه، موده، لارښوونې، ټیمپلیټ فایل، سر/پښه، فعال، اصلي).
- **کاپي** — د ټیمپلیټ یوه کاپي د نوې پاڼې په توګه جوړوي.
- **حذف** — تایید ډیالوګ خلاصوي؛ تایید کول ټیمپلیټ حذفوي.

### ډله ییز عملیات

په دې پاڼه کې ډله ییز عملیات شتون نلري.

---

## د نوي ټیمپلیټ جوړول

د نوي ازموینې پاڼې ټیمپلیټ جوړولو لپاره، په د ازموینې پاڼې ټب کې **"Create Template" (ټیمپلیټ جوړول)** باندې کلیک وکړئ. یو فورم خلاصیږي له:

### د فورم ساحې

| ساحه | ډول | اړین | توضیحات |
|------|-----|------|----------|
| مکتب | انتخاب یا لوستلو-یوازې | هو | مکتب (که اصلي مکتب وي لوستلو-یوازې). |
| تعلیمي کال | انتخاب | هو | تعلیمي کال. |
| ټولګی | انتخاب | هو | ټولګی (د ټولګي تعلیمي کال). |
| مضمون | انتخاب | هو | د ټولګي لپاره مضمون. |
| ازموینه (اختیاري) | انتخاب | نه | ازموینه یا "Generic Template (no exam)". |
| سرلیک | متن | هو | د ټیمپلیټ سرلیک. |
| ژبه | انتخاب | هو | انګلیسي، پښتو، فارسي، یا عربي. |
| موده (دقیقې) | نمبر | هو | ۱–۶۰۰. |
| ټولې نمرې (اختیاري) | نمبر | نه | اختیاري؛ نور له پوښتنو څخه اتوماتیک محاسبه کیږي. |
| لارښوونې | ټیکس اریا | نه | د زده کوونکو لپاره لارښوونې. |
| ټیمپلیټ فایل (اختیاري) | انتخاب + Manage Template Files | نه | ټیمپلیټ فایل وټاکئ یا اصلي وکاروئ. |
| Header HTML | ټیکس اریا | نه | دودیز سر HTML. |
| Footer HTML | ټیکس اریا | نه | دودیز پښه HTML. |
| فعال | سوئیچ | هو | اصلي په. |
| د ازموینې مضمون لپاره اصلي | سوئیچ | نه | د دې ازموینې مضمون لپاره اصلي په توګه نښه کړئ. |

### د ثبتولو وروسته څه کیږي

په بریالیتوب کې ډیالوګ بند کیږي او جدول تازه کیږي. د نوې ټیمپلیټ لپاره پوښتنې اضافه کولو له قطار مینو څخه **پوښتنې سمول** وکاروئ.

---

## د ټیمپلیټ سمون

۱. په جدول کې د عملیاتو مینو (⋮) → **سمون** باندې کلیک وکړئ.
۲. په فورم کې ساحې تازه کړئ.
۳. **"Update" (تازه کول)** باندې کلیک وکړئ. ډیالوګ بند کیږي او جدول تازه کیږي.

د پاڼې دننه پوښتنې بدلولو لپاره، پرځای یې له قطار مینو څخه **پوښتنې سمول** وکاروئ.

---

## د ټیمپلیټ حذف

۱. د عملیاتو مینو (⋮) → **حذف** باندې کلیک وکړئ.
۲. په ډیالوګ کې تایید وکړئ. ټیمپلیټ او د هغې آیټمونه حذف کیږي؛ دا بیرته اخیستل کېدی نشي.

---

## د ټیمپلیټ فایلونو ټب

**د ټیمپلیټ فایلونو** ټب د HTML ټیمپلیټ فایلونو لپاره یو مدیر ښیي. تاسو کولی شئ ټیمپلیټ فایلونه جوړ، سم او حذف کړئ. هر فایل نوم، ژبه او مینځپانګه لري. د ازموینې پاڼې ټیمپلیټ جوړولو یا سمولو وخت، په فورم کې **Manage Template Files** تڼۍ ورته مدیر په ډیالوګ کې خلاصوي ترڅو تاسو د هغې پاڼې لپاره ټیمپلیټ فایل وټاکئ. په ډیالوګ کې فایل ټاکل په فورم باندې ټاکي او ډیالوګ بندوي.

---

## د صادرولو اختیارونه

کله چې په فلټر شوي لست کې پاڼې ټیمپلیټونه وي، په د ازموینې پاڼې ټب په سر کې **Report Export** تڼۍ (PDF او Excel) ښکاري. صادرول اوسني فلټرونه او کالمونه لکه سرلیک، مضمون، ازموینه، ټولګی، ژبه، موده، پوښتنې، ټولې نمرې او حالت شاملوي. د راپور سرلیک "Exam Papers" دی او د فلټرونو لنډیز شامل دی.

---

## لارښوونې او غوره کړنې

- هر ژبې لپاره یو ټیمپلیټ فایل وکاروئ (لکه پښتو، عربي) ترڅو په هغې ژبه کې ټولې پاڼې ورته layout شریکې کړئ.
- د هرې ازموینې او مضمون لپاره چې تاسو کاروئ **د ازموینې مضمون لپاره اصلي** ټاکئ.
- بدلونونو (لکه مختلف لارښوونې یا موده) لپاره ټولې ساحې بیا دننه نه کولو **کاپي** وکاروئ.
- د ټیمپلیټ جوړولو وروسته، تل **پوښتنې سمول** خلاص کړئ ترڅو له پوښتنو ذخیرې څخه پوښتنې اضافه کړئ.
