# ټولګی–مضمون مارک شیټ

د ټولګي–مضمون مارک شیټ راپور د **یو مضمون** د **یو ټولګي** لپاره د ازموینې پایلې ښایي: د هر زده کوونکی نمرې د هغه مضمون لپاره، رتبه، سلنه، درجه او پایله (پاس/فیل/غیرحاضر). هغه کارکوونکي چې د ازموینې راپور اجازه لري دا د مضمون له مخې لستونه چاپ یا صادرولو، غوره اجرا کوونکو لیدلو، او لنډیز احصائیو (ټول زده کوونکي، د ټولګي اوسط، لوړې/ټیټې نمرې، د پاس سلنه) لیدلو لپاره کاروي. تاسو کولی شئ یو وخت یو ټولګی او یو مضمون وګورئ (واحد مضمون ټب) یا ټول ټولګي او ټول مضمونونه په یو فایل کې صادر کړئ (ډیر مضمونونه ټب).

---

## د پاڼې لنډیز

کله چې تاسو دا پاڼه خلاصه کړئ، تاسو به ووینئ:

### لنډیز کارډونه

لنډیز کارډونه **د راپور دننه** ښکاري وروسته له دې چې تاسو ازموینه، ټولګی او مضمون وټاکئ (په واحد مضمون لید کې).

- **ټول زده کوونکي** — په دې مضمون/ازموینه کې د ټولګي د زده کوونکو شمیر.
- **د ټولګي اوسط** — په دې مضمون کې ترلاسه شوې اوسط نمرې.
- **لوړې نمرې** — ترلاسه شوې لوړې نمرې.
- **ټیټې نمرې** — ترلاسه شوې ټیټې نمرې.
- **د پاس سلنه** — د هغه زده کوونکو سلنه چې په دې مضمون کې پاس شوي.

### فلټرونه او لټون

- **ازموینه وټاکئ** — د ازموینې ټاکلو لپاره کومبوکس. امکان وي چې اصلاً د اوسني تعلیمي کال وروستۍ ازموینه وټاکي.
- **ټولګی وټاکئ** — په واحد مضمون ټب کې ښودل کیږي. د ازموینې ټولګیو څخه یو ټولګی (او برخه) وټاکئ. مخکې له مضمون لست لوډیدو اړین.
- **مضمون وټاکئ** — په واحد مضمون ټب کې ښودل کیږي. د ټاکل شوي ټولګي لپاره د دې ازموینې ته ټاکل شویو مضامینو څخه یو مضمون وټاکئ.

آزاد متن لټون نشته. فلټرنگ یوازې د ازموینې، ټولګي او مضمون له مخې دی.

## ډاټا جدول

اصلي جدول (واحد مضمون لید، د ازموینې، ټولګي او مضمون ټاکلو وروسته) ښایي:

| کالم | توضیح |
|------|--------|
| رتبه | د نمرو له مخې د زده کوونکی رتبه (لوړترین لومړی). لومړی درې ممکن ټروفي آیکونونه وښایي. |
| عکس | د زده کوونکی عکس ټمبرنیل یا پلیس هولډر. |
| د زده کوونکی نوم | بشپړ نوم. |
| حاضري نمبر | د ازموینې حاضري نمبر. |
| د شمولیت نمبر | د شمولیت نمبر. |
| د پلار نوم | د پلار نوم. |
| ټولګی | د ټولګي نوم او برخه (په دې راپور کې د ټولو قطارونو لپاره ورته). |
| مضمون | د مضمون نوم (ټولو قطارونو لپاره ورته). |
| ترلاسه شوې نمرې | ترلاسه شوې نمرې، یا "غیرحاضر" بج، یا که دننه نه دي "-". |
| ټولې نمرې | د مضمون لپاره ټولې نمرې. |
| د پاس نمرې | د مضمون لپاره د پاس نمرې. |
| سلنه | د دې مضمون لپاره سلنه. |
| درجه | د درجو جدول څخه درجه. |
| پایله | پاس (شین)، فیل (سور)، یا غیرحاضر (آؤټلاین). |

جدول د لاندې کنټرولونو سره پاګینیټ شوی دی.

### غوره اجرا کوونکي

لنډیز لاندې، "غوره اجرا کوونکي" کارډ د نمرو له مخې لومړی درې زده کوونکي ښایي (د ټروفي آیکونونو او نمرو/سلنې سره).

### ټبونه

- **واحد مضمون** — ازموینه، ټولګی او مضمون وټاکئ. د هغه ټولګي لپاره د مضمون مارک شیټ ښودل کیږي. صادرول/چاپ د دې یو ټولګي–مضمون ترکیب پورې اړوند دي.
- **ډیر مضمونونه** — د ازموینې ټاکلو وروسته، تاسو کولی شئ پورته صادرول تڼۍ وکاروئ ترڅو یو فایل جوړ کړئ چې ټول ټولګي او ټول مضمونونه ولري (هر ټولګی–مضمون جلا Excel شیټ یا PDF برخه کې). دلته په هر مضمون جدول ټبونه نشته؛ پاڼه توضیح کوي چې صادرول تڼۍ وکاروئ.

### د قطار عملیات

په جدول کې د قطار عملیات نشته.

### ډله ییز عملیات

ډله ییز عملیات شتون نلري.

## صادرول اختیارونه

- **چاپ** — د MultiSectionReportExportButtons له لارې. واحد مضمون: یوه برخه. ډیر مضمونونه: ټول ټولګی–مضمون ترکیبونه د پاڼې ماتیدو سره.
- **PDF / Excel** — په واحد مضمون لید کې، کله چې راپور ډاټا ولري، MultiSectionReportExportButtons (او اختیاري ReportExportButtons) د فلټر لنډیز سره (ازموینه، ټولګی، مضمون، تعلیمي کال) PDF/Excel صادرول اجازه ورکوي. په ډیرو مضامینو لید کې، MultiSectionReportExportButtons د هغه ازموینې هر ټولګي او هر مضمون لپاره برخې جوړوي او صادروي (Excel: هر ټولګی–مضمون یو شیټ؛ PDF: د پاڼې ماتیدو سره برخې). کارول شوی ټیمپلیټ ډول: `class_subject_academic_year`.

صادرولونه اوسني ټب او ټاکل درکوي (واحد ټولګی–مضمون یا ټول ټولګي او مضمونونه).

---

## لارښوونې او غوره کړنې

- **واحد مضمون** وکاروئ کله چې تاسو د یو ټولګي په یو مضمون کې پاک لست اړین لرئ (لکه د مضمون ښوونکی سره شریکولو یا اعلان بورډ لپاره).
- **ډیر مضمونونه** صادرول وکاروئ کله چې تاسو ته د ټولې ازموینې لپاره د مضمون له مخې پاڼو بشپړ سیټ په یو فایل کې اړین دی (ټول ټولګي، ټول مضمونونه).
- که مضمون په لست کې ونه څرګندېږي، ممکن د ټاکل شوي ټولګي لپاره د دې ازموینې ته ټاکل شوی نه وي؛ لومړی د ازموینې ټولګی–مضمون setup وګورئ.
- "مضمون په ټولګي کې نشته" پیغام په ډیرو مضامینو ټب کې ښکاري کله چې ټولګي هغه مضمون ونلري.
